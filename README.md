# Real Estate Portfolio OS — Phase 1 (Foundation & Visibility)

## Goal
Create a single source of truth for portfolio docs, obligations, vendors, and issues — with a repeatable operating cadence.

## Portfolio
- A / Residential — Murrieta: 31685 Brisa Ct, Murrieta, CA 92563
- B / Condo — Vista: 980 Lupine Hills Dr Unit 82, Vista, CA 92081

## How to use
1. Open `portfolio-os.html` in your browser.
2. Store documents in `02-docs/` and in each asset’s `docs/` folder.
3. Track vendors in `04-vendors/vendors.csv`.
4. Track issues in `05-issues/issues.csv`.
5. Keep notes in `00-admin/notes/notes.md`.

## Naming standard (recommended)
[ASSET]-[CATEGORY]-[YYYY-MM-DD]-[DESCRIPTION]-v1.pdf

Example:
A-INSURANCE-2026-03-02-DeclarationPage-v1.pdf
